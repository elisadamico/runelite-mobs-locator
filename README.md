## How to Use

1. Open the plugin configuration
2. Type an NPC name in the "Search for Mob" field (e.g., "goblin", "cow", "guard")
3. Click on the game screen to activate
4. View spawn locations in the overlay panel and on the world map

## Configuration

- **Search for Mob** - Enter the name of any NPC to search for
- **Hull Highlight Color** - Customize the color of NPC outlines
- **Minimap Dot Color** - Choose the color for minimap markers
- **Panel Background Color** - Adjust overlay transparency
- **Show/Hide Features** - Toggle minimap dots and search results

## Search Tips

- Search is case-insensitive and uses partial matching
- "guard" will find "Guard", "Palace Guard", "Tower Guard", etc.
- Works with any NPC that has spawn data

Perfect for slayer tasks, PKing, questing, or just exploring Gielinor!
